# Fire U
Portable codes between different deep learning framework like `Pytorch`, `Tensorflow` etc.


### Architecture
```sh
fireu/
└── data
    ├── coco
    ├── preprocess
    └── utils
```


### dependences
+ Cython
+ opencv
+ numpy
+ scipy
+ matplotlib
+ pycocotools
