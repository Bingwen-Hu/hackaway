### note on gdb
```
 r run the program
 b [line number] set breakpoint
 dis [breakpoint number] disable breakpoint
 c [continue run]
 bt see the backtrace
 f [frame number] move to frame
 disp [variable] auto display 
 undisp [number] disable auto display
 p [variable] print value of variable
 p *d@10 print 10 elements for array d
 info [breakpoint/disp/vars] show infos
 l [function name] list function code
 n next line
```
